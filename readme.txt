=== WP List Posts Shortcode ===
Contributors: michaelha
Tags: posts, shortcode, list
Requires at least: 2.8
Tested up to: 3.3.1
Stable tag: 0.2

Adds a shortcode [wp_list_posts]. For example [wp_list_posts category=3 numberposts=7] will return an unordered list of the most recent 7 posts in the category ID 3.

== Installation ==

1. Download the plugin
2. Unzip to the 'wp-content/plugins' directory
3. Activate the plugin

== Usage ==
This shortcode can be rendered inside a post, page or even in a widget area. Refer to the template tag http://codex.wordpress.org/Template_Tags/get_posts for input parameter.
